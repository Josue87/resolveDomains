# resolveDomains

Given a list of domains, you resolve them and get the IP addresses.

# Installation

```
> git clone https://github.com/Josue87/resolveDomains.git
> cd resolveDomains
> go build
```

# Usage

```
> ./resolveDomains -d domainFiles.txt [-t 150] [-r 8.8.8.8:53]
```
